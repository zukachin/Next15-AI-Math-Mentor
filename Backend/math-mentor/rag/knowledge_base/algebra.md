# Algebra – Formulas and Identities

## Quadratic equations
- Standard form: ax² + bx + c = 0, a ≠ 0.
- Quadratic formula: x = (-b ± √(b² - 4ac)) / (2a).
- Discriminant D = b² - 4ac: D > 0 two distinct real roots; D = 0 one repeated root; D < 0 no real roots (complex roots).
- Sum of roots = -b/a; product of roots = c/a.

## Key identities
- a² - b² = (a - b)(a + b).
- (a + b)² = a² + 2ab + b²; (a - b)² = a² - 2ab + b².
- (a + b)³ = a³ + 3a²b + 3ab² + b³; (a - b)³ = a³ - 3a²b + 3ab² - b³.
- a³ + b³ = (a + b)(a² - ab + b²); a³ - b³ = (a - b)(a² + ab + b²).

## Linear equations and inequalities
- Linear equation: ax + b = 0 ⇒ x = -b/a (a ≠ 0).
- When multiplying or dividing an inequality by a negative number, reverse the inequality sign.
- Domain: denominator ≠ 0; expression under √ must be ≥ 0 when working in reals.

# Algebra – Solution templates (JEE-style)

## Quadratic equation
1. Write in standard form ax² + bx + c = 0.
2. Compute discriminant D = b² - 4ac; state nature of roots from D.
3. Apply quadratic formula or factorise if obvious; give both roots if D ≥ 0.
4. Check: substitute roots back or use sum/product relations.

## Equation involving √ or modulus
1. Identify domain: for √(f(x)), require f(x) ≥ 0; for 1/g(x), require g(x) ≠ 0.
2. Square both sides only when both sides are non-negative (or use |A| = B ⇒ A² = B² with care); for |f(x)| = g(x), split into f(x) = g(x) and f(x) = -g(x) and enforce g(x) ≥ 0.
3. Solve and reject any solution outside the domain.

## Word problem to equation
1. Define variable(s) clearly (e.g. x = number).
2. Translate conditions into one or more equations/inequalities.
3. Solve; interpret answer in context and check units.

# Algebra – Common mistakes and pitfalls

- Forgetting a ≠ 0 when using quadratic formula or dividing by a coefficient.
- Sign errors when expanding (a - b)² or (a + b)(a - b); (a - b)² = a² - 2ab + b² (not +2ab).
- Squaring an equation without checking sign: √x = -2 has no real solution; squaring gives x = 4, which is extraneous.
- Dividing both sides by an expression that could be zero (e.g. x) and losing the root x = 0.
- In inequalities, forgetting to flip the inequality when multiplying or dividing by a negative.
- Confusing “and” vs “or” when combining conditions (e.g. domain of 1/√(x-1) is x > 1, not x ≥ 1).

# Algebra – Domain and constraints

- Rational: denominator ≠ 0. Example: 1/(x² - 1) requires x ≠ ±1.
- Square root (reals): argument ≥ 0. Example: √(2x - 3) requires x ≥ 3/2.
- Both: for √(1/(x-2)), need x - 2 > 0 so x > 2.
- Modulus: |f(x)| is defined for all x; equations |f(x)| = g(x) need g(x) ≥ 0.
- When solving, always state or check domain and discard extraneous solutions.
