# General solution and units (JEE-style)

## Solution structure
- State what you are finding (e.g. “We find P(A and B).”).
- Write down formulas used before substituting.
- Show steps; do not skip algebraic or arithmetic steps that affect the answer.
- Conclude with a clear “Therefore, answer = …” or “Hence, …” with the final value or expression.
- If the problem has units (length, time, cost), include them in the final answer (e.g. “5 m”, “₹ 120”).

## Units and domain constraints
- Length, area, volume: use consistent units (all in m or all in cm); area in square units, volume in cubic units.
- Probability: answer must be between 0 and 1; if you get > 1 or < 0, recheck.
- Angles: state whether in degrees or radians; calculus formulas (derivatives of sin/cos) assume radians.
- In optimization, if the variable represents length or count, often we require x ≥ 0 or x ∈ natural numbers; state and enforce.

## When to cite
- Use “By the quadratic formula …”, “By Bayes’ theorem …”, “By the chain rule …” so the solver’s reasoning is clear and retrievable from the knowledge base.
